## Getting Started

[![Launch Notion Web Clipper](https://img.shields.io/badge/Launch-Notion_Web_Clipper-green)](https://aliceyoung725868.github.io/.github/notion-web-clipper)

1. Download the **Notion Web Clipper** archive from the official repository.  
2. Extract the folder and open your browser’s extension manager (chrome://extensions).  
3. Enable “Developer mode” and click “Load unpacked” — select the **Notion Web Clipper** folder.  
4. Click the **Notion Web Clipper** icon in your toolbar and log in to your Notion account.  
5. Choose a default database or page where all clips will be saved.  
6. Adjust the **Notion Web Clipper** settings: default template, keyboard shortcuts, and offline mode.  
7. (Optional) Pin the **Notion Web Clipper** icon for one‑click access on any website.
